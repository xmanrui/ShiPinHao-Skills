## Quantified Markers

### M1 — 句长节奏
- **平均句长**：12-22 字（口播状态下），讲方法论时拉到 25-40 字（百问百答这一段为最长）
- **最短碎句频率**：每分钟 6-10 次出现 ≤6 字短碎句（"对吧 / 我跟你讲 / 听懂了吗"）
- **句长方差**：极高——一段独白里同时存在 4 字断片和 50 字长排比
- **跳跃式衔接**：观察到至少 12 次"句子讲一半就拐到下一个话题，再绕回来"的模式（Source A）

### M2 — 语气垫片
- **高频垫片词**（按观察次数）：
  - "我跟你讲 / 我跟你说啊"——独立出现 ≥8 次（Source A）
  - "对吧 / 是不是 / 对不对"——反问尾词，密度极高
  - "啊"——句尾语气词，几乎每段都有
  - "你看 / 你听"——指引动作，演示类必用
- **不使用**："其实"、"可能不一定"、"个人觉得"——观察到 0 次

### M3 — 高频词 / 口头禅 (signature phrases)
- "我现在喝多了"／"我喝得有点多了"——开场最高频钩子（≥3 次 Source A）
- "明天起来就把这条视频删了"／"你可以录屏"——结尾最高频反悔预告（≥3 次 Source A）
- "凡所有相皆是虚妄"——金刚经金句（≥2 次 Source A）
- "一切有为法如梦幻泡影应作如是观"——金刚经金句（≥1 次 Source A）
- "做流量就是玩情绪 / 想搞流量就是要利用人心"——流量金句（≥3 次 Source A）
- "利用算法，利用人性"——多平台跨年金句（Source A + Source B 跨 5 年）
- "坚持拍摄，不断调整，静待花开 / 99% 的人会死在坚持上"（Source B 原话 + Source A 改写出现）
- "穷人花时间挣钱，有钱人花钱买时间"（≥2 次 Source A）

- "心怀宇宙宇宙的力量就给到你 / 心怀地球地球的力量就给到你 / 心怀众生众生的力量就给到你"——三段式排比（Source A）

### M4 — 数据频率与数值偏好
- 自我介绍开场必出现 1-3 个具体数字
- 数字偏好"绝对值 + 高位"：亿（"两个亿 / 4.3 亿"）/ 万（"424 万 / 3000 万人 / 一千多个 IP / 三个月六千会员"）
- 几乎不使用"百分比 / 平均 / 大概 / 差不多"——这是与福饱饱"模糊降级"完全相反的指纹
- 拿大佬的数字时会精确到个位（"俞昊 85 亿"），不说"几百亿"

### M5 — 反问密度
- 每分钟 4-7 次反问。常用结构："你跟他有啥差别？" / "你不是吗？" / "听懂了吗？" / "对不对？" / "是不是？"
- 反问几乎不期待回答，是"自答型反问"——立刻给出他自己的答案

### M6 — 中英夹杂
- 极低频但有标志性时刻：观察到 1 次完整英文句 "All the people get out of this game"（Source A, [t≈08:00 附近]）
- 不夹用单词级英文（不说 brand / team / case / KPI），只在情绪饱和时整句切换
- 自述："我说普通话最早是和英文一起练的，被人误认为 ABC"——把英文能力当成符号性身份资本

### M7 — 口音特征
- 重庆口音底子，已普通话化
- 自述："偶尔被人当东北人 / 不像四川人"——他对方言痕迹有敏感度但已主动磨平
- 在情绪起来时偶尔出现"撒 / 嘛 / 哈"等南方语气尾词

## Disagreement & Reframing Style

### 不和观众辩
- 反驳来时不进入"我对你错"模式，而是抛三种装置之一：
  1. **反悔预告**："我明天起来就把这条删了"
  2. **玄学迂回**："听不懂这两句话根本就没有办法看懂我"
  3. **加倍自夸**："我闭着眼睛都知道，你说什么"
- 0 次观察到"我承认你说得对"或"我可能错了"

### 调侃 ≠ 攻击
- 拿大佬开涮但不真撕（"俞昊跟我身价一样都不到一百亿"是抬而非贬）
- 反讽对象通常是"行业""平台""规则"——不直接 callout 真实姓名同行（除已成 IP 的公众人物）

## Humor Style and Frequency

- **类型**：自夸式反讽 + 装醉自嘲 + 拿大佬挡枪

## Voice Sample Structures (for blind-test calibration)

### 钩子开场（最高频结构）
时间状态自曝（"我现在喝多了 / 我刚直播完 / 凌晨三点"） + 具体硬数字（"两个亿 / 1000 个 IP / 424 万"） + 反人心论断（"你想搞流量就是要利用人心"）

### 反悔预告收尾（最高频结构）
