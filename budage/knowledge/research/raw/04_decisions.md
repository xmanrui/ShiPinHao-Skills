## Major Bets (按时间序)

### B4.1 — 2010：从大学毕业进入新东方
- **决策**：放弃专业方向（信息提示但未明确）进入英语教培行业
- **逻辑**：教培是当时高增长高回报赛道（Source B 间接交代）
- **结果**：成为新东方管理层，建立行业人脉资本

### B4.2 — 2015：从公众号转向音频（喜马拉雅）
- **决策**：内容形式切换
- **触发**：判断"长内容时代结束 / 音频是新机会"
- **结果**：未公开报道单独成果，但建立了"对内容形式敏感"的判断习惯

### B4.3 — 2018 夏：从长内容转向短视频（抖音）
- **决策**：彻底切到抖音
- **触发**："看到樊登读书在抖音矩阵账号粉丝超越 1 亿时，覃流星知道新的机会来了"（Source C 编辑表述，但反映他自述）
- **结果**：2019 年开始英语短视频 IP 创业，2020 年获蓝象资本天使轮

### B4.4 — 2020-03：天使轮融资 + 公开发布"短视频时代教育人位置"宣言
- **决策**：用一篇 36 氪首发把"超级名师收入超明星"做成行业 narrative
- **逻辑**：先建立行业话语权，再筹钱（Source C）
- **结果**：4 个月签约 100+ 名师，孵化数百个账号

### B4.5 — 2020 全年："沉迷于做增量"（自述失败）
- **决策**：把 KOL 粉丝量当主指标
- **结果**：本人在 Source B 中明确说"效率低下 / 走过弯路"，2021 转向变现
- **学到**：粉丝量 ≠ 商业价值，目标用户精准度胜于粉丝总量

### B4.6 — 2020-06：千万 Pre-A 轮（百度风投 + 蓝象 + 真格、字节）
- **决策**：扩大孵化规模 + "名师合作 + 素人孵化"双线
- **结果**：进入"网红校长"叙事高峰期（Source D, Source E）

### B4.7 — 2021 初：从粉丝增量转向变现
- **决策**："80% 精力做变现"
- **理由**：意识到投资人和团队需要兑现商业故事
- **结果**：2021-12 多知网采访时他自述"做了一天直播单日变现 424 万"

### B4.8 — 2024 前后（具体日期未明）：卖掉公司
- **决策**：将 101 名师工厂出售（音频中自述"卖了 4.3 亿"，但音频也显示他对此自我披露态度暧昧——既要甩数字，又怕被追问细节）
- **结果**：手握退出资金，开始下一阶段
- **可疑度标记**：4.3 亿数字仅来源音频自述，无第三方报道交叉验证 → *low confidence*

### B4.9 — 2025 中：发布"火种计划"
- **决策**：从教育/IP 孵化身份切换到心灵成长 IP
- **触发**：自述"美国老中医把脉 / 大哥说你拒绝使用自己的天赋 / 我厌倦了网红校长这四个字"
- **结果**：3 个月六千会员（Source A 自述，无外部验证 → *medium confidence*）

### B4.10 — 2026 初：开"不答哥"分身做短视频教学
- **决策**：在阿星 IP 之外再开一个分身专做"流量教学"
- **逻辑**：把心灵成长（高单价 / 低争议）和 IP 教学（高争议 / 高复购）赛道分开
- **结果**：当前主推（音频中自述"我现在要做不答哥"）

## Reversals (改变方向的时刻)

### R4.1 — 2021：从"做粉丝量"转"做变现"
- **触发**：投资人和现金压力
- **从**：粉丝指标
- **到**：商业定位指标 + 变现指标
- **代价**：放弃了部分纯做大体量的账号
- **Note**：他承认这个转向迟了，本应更早

### R4.2 — 2025：从"网红校长"转"修行 / 火种"
- **触发**：自述"被点拨 + 厌倦"，但旁观推断也包含"教培行业被严打"和"IP 孵化赛道竞争白热化"两个外因
- **从**：教培 IP 孵化
- **到**：心灵成长 / 修行
- **代价**：放弃"网红校长"已建立的品牌资本（但保留为简历背书）

### R4.3 — 2026：从"阿星单 IP"转"阿星 + 不答哥双 IP"
- **触发**：火种计划锁定中产女性，但流量教学市场更大
- **从**：单赛道
- **到**：双分身（修行 / 流量）
- **代价**：人设连贯性受质疑（"你说我们哪有啥人设"是这个 reversal 的 hedge）

## Tradeoffs（明确的牺牲选择）

### T4.1 — 选"金句密度"而非"完整论证"
- 他公开内容几乎不做 5 分钟以上的连续推理；所有"系统思考"压缩为 30 秒可截图金句
- **牺牲**：深度受众
- **换取**：传播率 + 短视频钩子适配性

### T4.2 — 选"自夸式信用"而非"案例式信用"
- 他不公开学员的具体收入数据（除少数有 IP 的样板生）
- **牺牲**：第三方可验证性
- **换取**："1000 个 IP / 一天 424 万"这种总和数字的话语权

### T4.3 — 选"目标用户精准"而非"粉丝量大"
- 自述："目标用户精准度胜于粉丝总量"
- **牺牲**：粉丝总量
- **换取**：变现率
- **Note**：这是他 2021 年的转向心得

### T4.4 — 选"佛学话术保护层"而非"直接商业话术"
- 火种计划用"小喇叭 / 觉醒 / 火种"语言，不直接说"高客单价心灵课"
- **牺牲**：表达直接性
- **换取**：避免"卖课割韭菜"的舆情火力

## Patterns and Repeated Themes
- **平台敏感性强**：每次平台/形式拐点都早于平均水平做切换（公众号→音频→短视频→直播→视频号）
- **决策驱动是"赛道升级"而非"垂直深耕"**：5 次身份切换，每次都换赛道而非加深现有
- **失败叙事都被转化为方法论资产**：他承认失败但永远附带"我学到了 X"
- **数字披露选择性极强**：大数披露（2 亿 / 4.3 亿 / 424 万）频繁，过程数据（毛利、续费、退课率）零披露

## Contradictions

- C4.2：自述"目标用户精准胜于粉丝量" vs. 自我介绍永远先甩"3000 万人 / 1000 个 IP"这种"粉丝量"

## Inferences (clearly marked)
- *Inference*：他的快速决策能力部分来自早期教培行业（高频招生压力 / 高频试错）的训练，不完全是天赋
- *Inference*："卖公司 4.3 亿"如属实，他当下的"做不答哥 + 火种"更像是个人意义寻找 + 二次创业品牌重塑，而非纯商业必要性
