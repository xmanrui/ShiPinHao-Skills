## Evidence

### E2.1 — 被追问"为什么这么自夸"时的标准应对
- 在直播中被嘉宾或观众试探"你这数字真的吗"时，他不解释、不补充证据，直接抛"我现在喝多了 / 明天起来我可能就把这条删了 / 你可以录屏"。（Source A, [t≈01:15–01:35] 和 [t≈03:50–04:10] 重复出现）
- 这是他的**逃生通道型 hedge**：用"反悔预告"换"大胆话许可"。
- **Pattern**：他不擅长（也不愿意）做"防守型澄清"，他的对话防御是"反向加冕"——把澄清变成更夸张的下一句。

### E2.2 — 被问到"网红校长这个身份"时的迂回
- 多知网记者问他怎么看"网红校长"称呼，他没有正面接，先讲"商业定位优先于个人定位"，再讲"流量敏感度"，最后才回到身份："我现在 80% 的精力在变现"。（Source B 段落转述）
- 音频中被同样话题触发时，他直接跳到"我厌倦了这四个字"——把回避包装成"觉醒"。（Source A, [t≈07:00–07:30]）
- **Pattern**：身份相关问题永远绕开正面回答。早期靠"商业话术"绕，2026 年靠"玄学话术"绕。

### E2.3 — 在直播间和嘉宾对谈时的主导节奏
- 与文绣师对谈片段：嘉宾报价"198 一次"，他立刻打断："你这个价格收得太低了 / 我作为视频号的顶流 / 木子都给我做眉毛了。"（Source A, [t≈22:30–23:20]）
- 整段对话他主导大约 70%——把嘉宾的故事 hook 成"审美 / 修行 / 商业升级"再回扣到自己的方法论。
- **Pattern**：嘉宾对话不是"问询"，是"借嘉宾话头讲自己的方法论"。

### E2.4 — 调侃大佬的固定模板
- 谈到俞昊（疑指俞敏洪 / 俞军 / 樊登 / 罗振宇某一人，音频未明确）："俞昊不是百亿大佬，跟我身价一样都不到一百亿。"（Source A, [t≈15:40–16:00]）
- 谈到马斯克："他能日更 100 条你也能日更 100 条 / 他天天装疯卖萌不也是为了挣钱吗？"（Source A, [t≈15:00–15:20]）
- 谈到库克："库克不是厨师吗？带个厨师干啥？"（Source A, [t≈15:25–15:40]）
- **Pattern**：调侃大佬永远是"先找到一个对位的可比项 + 再用调侃打掉距离感 + 最终把自己摆在和大佬同一档"，从不真撕。

### E2.5 — 被观众反驳时的"不答"
- 直播评论区有人质疑（音频未直接呈现观众文字，但能听到他读评论后的回应）："你说我们哪有啥人设？我现在喝多了，明天起来我可能就把这条视频删了。"（Source A, [t≈04:20–04:40]）
- "听不懂这两句话根本就没有办法看懂我"——把"不解释"包装成"你不配懂"。（Source A, [t≈20:35–20:55]）
- **Pattern**：反驳来时他从不进入辩论模式，要么"反悔预告"，要么"贬低提问者智识"。

### E2.6 — 在长访谈中被追问"失败"时
- 多知网记者问到失败案例（北大丁教授半年仅 400 粉），他没有回避，但把责任分配为"前 MCN 的方法不对"+"后期我们调整就起来了"——把失败转化为方法论的反例验证。（Source B）
- **Pattern**：他承认失败，但永远把失败重新框定为"方法论的成功证据"。

## Patterns and Repeated Themes
- **对话主导欲极强**：嘉宾对谈中他平均独白比例 60-70%，基本不让对方完整说一段
- **追问应对三板斧**：反悔预告（明天就删）/ 玄学迂回（觉醒了 / 你不配懂）/ 数据反压（"我闭着眼睛都知道"）
- **不进入正面辩论**：观察到 0 次"我承认你说得对"或"这点你说的有道理"
- **被夸时立刻自嘲挡牌**：每次自夸后必接"我喝多了"——给后面的话留逃生通道
- **大佬调侃用结构相同的模板**：找对位 → 调侃距离 → 自我摆位

## Contradictions

- C2.2：他承认"全年沉迷做增量是错的" vs. 音频里反复甩"3000 万人看见 / 1000 个 IP / 一天 424 万"——他对自己的"沉迷做增量"教训没有真正内化，只是修辞调整
