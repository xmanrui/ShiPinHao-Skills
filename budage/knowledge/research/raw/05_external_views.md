## Evidence — How Others Describe Him

### E5.1 — 媒体笔下的"野心 + 自然高级感"
- i黑马原文："身材小但气场强大 / 自信热情 / 自然的高级感 / 不是一般的创业者"（Source A）
- 这与他自己音频里"我现在喝多了"的醉酒人设形成 contrast——**外部视角看到的是自信精英，他自己外推时刻意降级为江湖醉汉**

### E5.2 — 业内对"29.8 万短视频培训"的两极反应
- 早期质疑："覃流星在割有钱人的韭菜 / 新版拼单名媛 / 视频内容真实性受质疑"（Source A、Source B）
- 中期转变（i黑马原话）："质疑逐渐转变为惊叹，再到沉默"（Source A）
- 他本人的回应："我割有钱人、劫富济贫怎么了？"（Source B 直接引用）
- **关键 reframing**：外部把"29.8 万"看作 controversy，他把它包装成"劫富济贫的道义高度"

### E5.3 — "椅子营销"作为外部识别符号
- Source B 报道："所有创始人都坐过的这把椅子 / 这把椅子是视觉锤，而 29 万 8 就是文字钉"
- 他自己在音频中也甩这个梗：（音频内"爱马仕橙的椅子每天能被三千万人看见"，[t≈07:00 附近]）
- **重要 cross-context 点**：椅子是他和外界的共同符号——他承认有营销设计，外界也用这个符号识别他。这是他**最少 hedge 的真话**。

### E5.4 — "16 字方法论"被业内引用
- i黑马报道他的方法论："专家人设，出圈话题，满足好奇，打动人心"（Source A）
- 这个版本 2021 比 2026 音频中"利用算法 利用人性 / 玩情绪 / 100→30→10"更结构化、更冷静
- **Pattern**：他对媒体说的方法论（A 版本）是冷静框架；对镜头观众说的方法论（B 版本）是情绪话术——他的双口径

### E5.5 — 团队工作风格（隐性批评）
- "10106 工作节奏（10 点上班、10 点下班、周 6 天），虽称'不是公司要求'但'很少人愿意提前走'"（Source A）
- **暗批评信号**：i黑马以中性笔触呈现，但实际是高强度工作文化的标记
- 他自己在公开内容中从不主动谈 996 / 10106——这是被外部观察补出的盲点

### E5.6 — 风险点 1：政策风险
- i黑马指出："双减政策落地可能导致灭顶之灾"——尽管他通过 2B→2C 切换规避了，但这个判断暗示他的赛道脆弱性
- 他公开内容中**几乎不谈政策**——这是 stated 与 revealed 之间的关键 gap

### E5.7 — 风险点 2：成功源自客户筛选还是孵化能力
- i黑马原文："能支付 29 万 8 的客户本身就是精英，成功因素部分源于客户基础而非纯粹的孵化能力"（Source A）
- 这是外部视角对他**孵化能力归因谬误**的精准批评
- 他在音频中反复甩"1000 个 IP / 一天 424 万"作为孵化能力证据——但缺乏对照组（如果不孵化这些客户会怎样？）

### E5.8 — 风险点 3：失败案例转嫁
- 合伙人刘冠奇坦言失败案例中"个人的想法比较多，不太听我们的建议"（Source A）
- 这是 2021 年的失败叙事框架——外部视角能看到失败被外推到学员身上
- 他自己在音频中有同款叙事："99% 的人会死在坚持上"——把孵化失败 100% 归到学员

### E5.9 — 风险点 4：精英主义倾向
- i黑马："覃流星并不会在平庸的人身上浪费时间"（Source A）
- 与他对镜头说的"姐妹们 / 火种宝宝们 / 给素人讲流量"形成 tension——**他对外是亲民布道，对内/对客户筛选是精英筛选**

### E5.10 — 财商教育批评（他孵化的学员所处赛道）
- 搜狐报道："财商教育平台存在无办学资质、虚假宣传、诱导投资、被投诉退费风险 / 讲师无官方资质认证"（Source B）
- 他孵化的 IP 中有财商类（如 Source B 提到的廖恒悦），他作为孵化方对学员合规性的责任在公开内容中未明确表态
- *Inference*：这是他后期转向"心灵成长 / 火种"赛道的隐性外因之一——心灵成长比财商教育的合规风险低

## Patterns and Repeated Themes
- **外部视角看到他自己不愿正面承认的三件事**：
  1. 客户筛选效应（不是孵化能力）
  2. 工作强度的高压团队文化
  3. 政策合规与赛道脆弱性
- **"质疑→惊叹→沉默"叙事**：i黑马的这个表述被他间接吸收为"忽略反对意见"的合理化模板
- **椅子营销**是他和外界共同认可的符号，少有的"无 hedge 的真话"

## Contradictions（外部视角揭示的与他自我叙事的 gap）
- C5.1：他对外说"商业定位优先于个人定位" vs. 外部视角看他是**先打造个人品牌再算商业**（i黑马 / 搜狐都把"网红校长"个人 IP 当作他成功核心）
- C5.2：他对外说"99% 的人死在坚持上"暗示自我严苛 vs. 外部视角发现他自己 5 年换了 5 个赛道，"换"才是他真正的策略
- C5.3：他对镜头是"亲民醉酒讲真话" vs. i黑马描述的是"自然高级感的精英创业者"——register 是表演性的
- C5.4：他对内容自述是"经过设计 / 出圈选题" vs. 镜头表演是"喝多了说真话"——他对**精心设计 vs. 即兴坦白**这两套叙事策略性混用
- C5.5：他的方法论 stated 是 16 字结构化框架，revealed 是"利用人心 / 玩情绪"

## Inferences (clearly marked)
- *Inference*：他的"椅子营销"被外界拆穿后他没有否认而是接受这个符号，说明**他对营销设计的暴露阈值很高**——他相信"人们知道你在营销也照样会买"
- *Inference*：从教培 IP 孵化转向心灵成长，部分是合规风险驱动（财商赛道高风险），他叙事中的"觉醒"是糖衣
- *Inference*：他公开内容中几乎不谈政策风险，可能是因为这是他的真盲点，也可能是有意回避——基于现有材料无法区分
