## Phase Timeline (按认知阶段非简历)

### Phase 1 — 教培管理者期（2010 → 2015）

**事件**：大学毕业入新东方，做到管理层；后转英语流利说做管理。

**当时认知**："教培行业是头部赛道，做好管理 = 走稳路径。"

**思维特征**：
- 体系化、组织化思维
- 信任头部品牌的"光环价值"
- 决策追随大平台 BD / KOL 的既定路径

**留下的资本**：
- 行业人脉（后来的合伙人刘冠奇、李石都来自这一圈）
- 对"教培用户购买路径"的肌肉记忆
- "新东方光环"——后来始终拿来背书

---

### Phase 2 — 内容形式敏感期（2015 → 2018）

**关键转折**：2015 年从公众号转音频（喜马拉雅）。

**认知转变**："长内容时代结束 / 形式比内容重要。"

**触发事件**：
- 看到公众号红利衰退
- 喜马拉雅 / 得到崛起

**思维特征出现**：
- 形成"对内容形式拐点的本能嗅觉"——这是他后来反复使用的元能力
- 第一次实践"早期信号 → 跨平台押注"的决策模板

**留下的资本**：
- 对"内容载体迁移"的信心和路径感
- 对"用户注意力分布变化"的提前感知机制

---

### Phase 3 — 短视频先发期（2018 → 2020）

**关键转折**：2018 年夏切入抖音。

**认知转变**："短视频是教育 + 名师的下一个 1 亿粉丝平台。"

**触发事件**：
- 看到樊登读书在抖音矩阵号粉丝过亿（"覃流星知道新的机会来了"——36 氪 2020 报道）
- 对"短视频 = 教育人重新被看见"的判断

**关键决策**：
- 2019 创立 101 名师工厂
- 2020-03 拿天使轮（蓝象资本）
- 2020-06 拿 Pre-A（百度风投 + 字节 + 真格 + 蓝象）

**思维特征**：
- 把"形式拐点"打成行业 narrative
- 用 36 氪首发宣言锁定话语权后再扩规模
- 数据敏感度建立："4 个月签约 100+ 名师，孵化数百个账号"成为标准开场

**未来 5 年都在沿用的金句**：
- "利用算法，利用人性"
- "坚持拍摄，不断调整，静待花开"
- "99% 的人会死在坚持拍摄这件事上"

---

### Phase 4 — 增量沉迷与反思期（2020 全年）

**自述失败**：（多知网 Source B）"最开始的时候就是追求粉丝数量 / 全年沉迷于做增量 / 效率低下"

**认知转变**：第一次显式承认"做大体量 ≠ 商业价值"。

**没说出口但留下指纹的判断**：
- 投资人和现金压力是触发反思的主因（外推但 2026 音频中暗示）
- 他对"承认错误"有承受力，但永远附加"我学到了 X"

**关键思维进化**：从"粉丝量优先"→"目标用户精准度优先"。

---

### Phase 5 — 变现转向 + 网红校长品牌期（2021）

**关键决策**：80% 精力做变现 / 单日直播 424 万。

**认知巩固**：
- "商业定位优先于个人定位"成型（虽然他自己实际操作仍是个人定位先）
- "29.8 万短视频培训"产品化
- "椅子营销 / 视觉锤 + 文字钉"成为他的标志符号

**此期外部叙事高峰**：
- i黑马、新浪、多知网长访谈集中在这一年
- 外部从"质疑→惊叹→沉默"——他对反对意见的合理化模板成型
- "16 字方法论：专家人设、出圈话题、满足好奇、打动人心"被业内引用

**重要 reframing**：
- 他承认"全年沉迷增量"是错的，但**没有承认"个人 IP 优先"是错的**——这是他和真正的"商业定位优先"派的关键分野

### Phase 6 — 政策避险与赛道转向期（2021 末 → 2024）

**外部触发**：
- "双减"政策落地（2021-07）—— i黑马原话："双减落地可能导致灭顶之灾"（Source C）
- 教培行业整体监管收紧
- 财商教育合规风险积累（Source B 报道）
- IP 孵化赛道竞争白热化

**他的应对**：
- 业务从 2B 向 2C 倾斜
- 最终（2024 前后）卖掉公司（自述 4.3 亿，无第三方验证）
- 这是他**第一次没用"觉醒"叙事**而是直接退出的转折

**未公开记录**：
- 卖公司细节（金额、买方、时间）
- 团队后续去向

**他自己叙事中刻意淡化这一阶段**——音频中关于"为什么离开"全部用"我厌倦了网红校长这四个字"覆盖

### Phase 7 — 心灵成长 / 火种计划期（2025）

**关键决策**：发布"火种计划"，从教育 IP 孵化转向心灵成长。

**触发叙事**（自述）：
- "我有一个大哥修行了很多年"
- "他给我推荐了一个美国老中医，把脉就能知前世"
- "他说你拒绝使用自己的天赋"
- "我厌倦了网红校长这四个字"

**触发的真实因素（推断）**：
- 教培监管 + 财商赛道合规压力
- IP 孵化方法论被复制，护城河变浅
- 高客单价（火种会员）+ 低争议（心灵成长）= 更稳赛道

**思维进化**：
- 把"赛道升级"包装成"觉醒"
- 把"商业 hedge"包装成"利他"——"我是小喇叭"
- 把"佛经金句"和"流量金句"缝合到同一个表达体——这是他的元能力延伸

**留下的资本**：
- "火种宝宝 / 小喇叭 / 心怀宇宙" 等火种语言库
- "三个月六千会员，妥妥行业头部"作为新阶段的 KPI 自证

### Phase 8 — 双 IP 期：阿星 + 不答哥（2026 至今）

**关键决策**：在阿星之外开"不答哥"分身，专做短视频教学赛道。

**叙事框架**：
- "我现在要做不答哥"
- "你说我们哪有啥人设"——把分身合理化为非人设
- "明天起来就把这条删了"——为大胆话留逃生通道

**思维特征**：
- 他重新进入"教学 + 流量"赛道，但用"不答哥"分身规避了"网红校长"的旧包袱
- "不答"作为防守 + 钩子的双重装置（详见 Dimension 2 / 3）
- 三分身随时切换——他对"人设 = 工具"的认识达到顶峰

**留下的判断（音频中）**：
- "做流量就是玩情绪 / 想搞流量就是要利用人心"——比 2021 表达更外露
- 数据上抬法（"2 个亿 / 1000 个 IP / 424 万 / 6000 会员"）成为开场必甩
- 大佬定位法（拿俞昊 / 马斯克 / 库克对位）是他这一阶段的新组合武器

## What He Used to Believe but No Longer Does

- **以前信**："粉丝量是 KOL 的核心资产" → **现在信**："目标用户精准度 + 变现转化 > 粉丝量"
- **以前信**："教育是好赛道" → **现在信**："心灵成长是更高单价 + 更低风险的赛道"（虽然不直说）
- **以前信**："网红校长是好身份" → **现在信**："分身是工具，不必锁死单一身份"
- **以前信**（2020-2021）："要 2B 服务大客户" → **现在信**："2C 直播 + 课程 + 会员 = 更稳现金流"

## Patterns Across the Trajectory

- **元能力稳定**：对"形式 / 平台拐点的嗅觉"是他 5 年内反复展示的核心能力，从公众号→音频→短视频→直播→分身策略
- **叙事转化机制稳定**：每次转向都用"觉醒 / 厌倦 / 找到新天命"叙事覆盖商业必要性
- **数据自证机制稳定**：每个新阶段都立刻给出一个"开口数据"作为新阶段成功证据（不验证，但也不假）
- **失败转嫁机制稳定**：把孵化失败 100% 归到学员（"99% 的人死在坚持上"）
